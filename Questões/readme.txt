Para executar os códigos, use uma IDE, no caso do IDLE.

Para a Questão 1, clique em arquivo, e depois em abrir, ou pelo atalho "CRTL+O", depois vá até a pasta onde está a pasta "Trabalho2", entre na pasta "Questao1" e selecione o arquivo "modela_complexo.py" e clique em abrir, pressione "F5" para executar o código, agora faça o mesmo processo para o código "Main.py", até executa-lo, os dados já foram inseridos no código, você pode ver como foram feitos pelo IDLE.

Para a Questão 2, clique em arquivo, e depois em abrir, ou pelo atalho "CRTL+O", depois vá até a pasta onde está a pasta "Trabalho2", entre na pasta "Questao2" e selecione o arquivo "cripto.py"  e clique em abrir, pressione "F5" para executar o código, agora faça o mesmo processo para o código "maincrip.py", até executa-lo. Primeiro vai ser solicitado um número, já criptografado para ser descriptografado, é só digitar algum número e apertar enter.(O método de criptografia também foi implementado).
